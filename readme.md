# 说明

主控 STM32F103C8T6 128Kb+20Kb 72MHz，开发环境CubeMx配置工程，IAR9.30.1开发环境，DAP-link仿真器

## 资源清单

| 类型     | IO            | 说明 |
| -------- | ------------- | ---- |
| MCU      | STM32F103C8T6 |      |
| 晶振     | 8M            |      |
| 光耦输入 | X0~X14        |      |
| 光耦输出 | Y0~Y10        |      |
| 模拟输入 | ADC           |      |
| 电机控制 | O11,O12       |      |
| 通信     | RS485         |      |
